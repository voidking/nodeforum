# 用到的软件和模块
下载安装Node  
https://nodejs.org/en/
## v0.0.0
```
npm install express  
npm install ejs   
npm install mongoose  
npm install bower -g  
bower install bootstrap  
```

## v0.0.1
下载安装MongoDB  
https://www.mongodb.org/downloads
